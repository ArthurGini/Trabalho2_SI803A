## Resumo:
Este trabalho descreve os procedimentos para analisar dados das músicas mais ouvidas do grupo Racionais MCs e do Emicida a partir da ótica da visualização de informação utilizando as técnicas de Word Clouds e Phrase Nets, descrevemos brevemente sobre os dados e demonstramos os gráficos comparando as fontes obtendo análises pertinentes sobre os dados.
